## Pre-Requisites

> Node.js 20.17.0
> https://nodejs.org/en/download/prebuilt-installer


## Installation

### Open terminal in /application/frontend

Install dependencies
```
npm install
```
<br>




## Running the Server

For development: start a development server
```
npm run start
```
<br>

For deployment: create a production-ready build
```
npm run build
```
